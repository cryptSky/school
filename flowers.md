## Крок 2. Запусти базовий код

Скопіюй цей код у Mu:

```python
from turtle import *
from random import random


def draw_flower():
    # --- стебло і листочки ---
    penup()
    goto(0, -150)
    left(90)
    pendown()

    # TODO: обери колір для стебла і листочків
    pensize(5)              # товщина стебла

    forward(50)
    circle(15)              # лівий листочок
    forward(50)
    right(180)
    circle(15)              # правий листочок
    right(180)
    forward(50)
    circle(15)              # верхній лівий листочок
    forward(80)

    # --- пелюстки ---
    pensize(1)              # повертаємо тонке перо
    right(90)
    # TODO: обери колір пелюсток
    for i in range(6):
        circle(20)
        left(60)

    # --- серединка ---
    # TODO: обери колір серединки
    circle(8)


draw_flower()
done()
```

Натисни ▶️ **Run**. Має з'явитись чорна квітка.

> 💡 Закривай вікно з малюнком перед кожним новим запуском.

> 🔍 **Що таке функція?** Усі команди малювання знаходяться всередині `def draw_flower():`. Це означає: «Python, запам'ятай, як малювати квітку». Команда `draw_flower()` в кінці — це наказ: «зараз намалюй». На наступному уроці ми використаємо цю функцію, щоб малювати багато квіток одним рядком!

# УРОК 2 — Зірка з квіток ⭐

На минулому уроці ти намалював одну квітку, яка росте з низу екрану. Сьогодні ми намалюємо **зірку** — 6 квіток, що ростуть з одної точки в усі боки!

```
       🌸
    🌸     🌸
       ⭐
    🌸     🌸
       🌸
```

## Крок 1. Модифікуємо функцію

Відкрий свій `flower.py` з минулого уроку. У ньому вже є функція `draw_flower()` — вона малює квітку, яка росте вгору з низу екрану.

Тепер нам треба, щоб квітка могла рости **в будь-якому напрямку**, починаючи з центру `(0, 0)`. Для цього зробимо дві зміни:

1. **Додамо параметр `angle`** — кут, у якому ростиме стебло.
2. **Замінимо `goto(0, -150)` на `goto(0, 0)`** — щоб всі квіти починались з центру.
3. **Замінимо `left(90)` на `setheading(angle)`** — повертаємось у напрямку, що передали.

**Видали увесь код** і встав цей оновлений варіант:

```python
from turtle import *
from random import random

speed(0)


def draw_flower(angle):
    # переходимо в центр і повертаємось у потрібному напрямку
    penup()
    goto(0, 0)
    setheading(angle)
    pendown()

    # стебло і листочки
    color("green")
    pensize(5)
    forward(50)
    circle(15)
    forward(50)
    right(180)
    circle(15)
    right(180)
    forward(50)
    circle(15)
    forward(80)

    # пелюстки
    pensize(1)
    right(90)
    color("red")
    for i in range(6):
        circle(20)
        left(60)

    # серединка
    color("yellow")
    circle(8)


# --- ТУТ БУДЕ ТВІЙ КОД ---
draw_flower(90)


done()
```

> 🔍 **Що нового?** Тепер функція приймає `angle` — кут стебла. У дужках при виклику `draw_flower(90)` ми кажемо: «намалюй квітку, що росте під кутом 90°» (тобто вгору).

▶️ Run. Має з'явитись одна квітка, що росте вгору з центру екрану.

## Крок 3. Що таке цикл?

А якщо хочемо **6 квіток** під різними кутами — нерозумно писати `draw_flower(...)` шість разів. Кращий спосіб — **цикл**:

```python
for i in range(6):
    print("Привіт!")
```

Це означає: «**6 разів** виконай те, що всередині». `i` — номер повторення (`0, 1, 2, 3, 4, 5`).

Ти вже бачив цикл — у функції `draw_flower` є:
```python
for i in range(6):
    circle(r)
    left(60)
```
Тут `6 разів` малюється коло з поворотом — так виходять пелюстки.

## Крок 4. Завдання — зірка з 6 квіток ⭐

Видали ті 4 рядки `draw_flower(...)` і напиши **цикл**:

```python
for i in range(6):
    angle = 90 + i * 60
    draw_flower(angle)
```

**Розгадай формулу:**
- При `i = 0` → `angle = 90` (квітка вгору)
- При `i = 1` → `angle = 150`
- При `i = 2` → `angle = 210`
- ...
- При `i = 5` → `angle = 390`, що те саме, що `30°`

Між квітками **60°**, а `360 / 6 = 60` — повне коло поділене на 6 рівних частин!

▶️ Run. Має з'явитись зірка з 6 квіток! ⭐

**Якщо щось не так:**
- `IndentationError` → перевір відступи. Обидва рядки всередині циклу мають бути зсунуті праворуч.
- Квітки розходяться нерівномірно → перевір, що в циклі `range(6)` і кут `i * 60`.

## Крок 5. Завдання — більше квіток у зірці

Спробуй зробити зірку з **8 квіток**. Що треба змінити?

- `range(6)` → `range(8)`
- `i * 60` → `i * 45` (бо `360 / 8 = 45`)

```python
for i in range(8):
    angle = 90 + i * 45
    draw_flower(angle)
```

А **12 квіток**? Підказка: `360 / 12 = 30`.

## Крок 6. Завдання — випадкові кольори 🎨

Зараз усі квітки одного кольору. Зробимо їх **різнокольоровими**!

### 6.1. Як працюють кольори в Python

Будь-який колір складається з трьох чисел: **скільки червоного, зеленого, синього** (RGB):

| Команда | Колір |
|---------|-------|
| `color(1, 0, 0)` | червоний |
| `color(0, 1, 0)` | зелений |
| `color(0, 0, 1)` | синій |
| `color(1, 1, 0)` | жовтий |
| `color(0.5, 0, 0.5)` | фіолетовий |

Числа від `0` (нічого) до `1` (максимум).

### 6.2. Команда `random()`

Команда `random()` дає випадкове число від 0 до 1 — кожного разу інше:
- `0.7234`, `0.0512`, `0.9981`...

Ми вже додали її на початку файлу: `from random import random`.

І заміни його на:

# 💡 Шпаргалка

### Команди turtle

| Команда | Що робить |
|---------|-----------|
| `forward(100)` | Уперед на 100 |
| `left(90)` / `right(90)` | Поворот ліворуч / праворуч |
| `setheading(90)` | Стати у напрямку 90° (вгору) |
| `circle(25)` | Коло радіусом 25 |
| `goto(x, y)` | Перейти у точку |
| `penup()` / `pendown()` | Підняти / опустити перо |
| `pensize(5)` | Товщина лінії |
| `color("red")` або `color(1, 0, 0)` | Змінити колір |
| `bgcolor("lightblue")` | Колір фону |
| `speed(0)` | Найшвидше |

### Кути в turtle

| Кут | Напрямок |
|-----|----------|
| `0` | праворуч → |
| `90` | вгору ↑ |
| `180` | ліворуч ← |
| `270` | вниз ↓ |

### Цикл і функція

```python
for i in range(6):
    angle = 90 + i * 60
    draw_flower(angle)
```

`i * 60` — рівномірно поділяє коло на 6 частин (`360 / 6 = 60`).
