# 🌸 Малюємо квітку і саджаємо сад

**Мета:** за два уроки створити свою програму, яка малює сад з різнокольоровими квітами.

**Інструмент:** Mu Editor.

**Файл:** один — `flower.py`. Працюватимеш у ньому весь час.

---

# УРОК 1 — Квітка 🌸

## Крок 1. Створи файл

1. Відкрий **Mu Editor**.
2. Натисни **«New»**.
3. **«Save»** → назва `flower.py`.

## Крок 2. Запусти базовий код

Скопіюй цей код у Mu:

```python
from turtle import *

# --- стебло і листочки ---
penup()
goto(0, -180)
left(90)
pendown()

# TODO: обери колір для стебла і листочків

forward(60)
circle(20)              # лівий листочок
forward(60)
right(180)
circle(20)              # правий листочок
right(180)
forward(60)
circle(20)              # верхній лівий листочок
forward(100)

# --- пелюстки ---
right(90)
# TODO: обери колір пелюсток
for i in range(6):
    circle(25)
    left(60)

# --- серединка ---
penup()
goto(0, 90)
# TODO: обери колір серединки
pendown()
circle(10)

done()
```

Натисни ▶️ **Run**. Має з'явитись чорна квітка.

> 💡 Закривай вікно з малюнком перед кожним новим запуском.

## Крок 3. Додай кольори

Команда зміни кольору:

```python
color("red")
```

| Колір | Команда |
|-------|---------|
| зелений | `color("green")` |
| червоний | `color("red")` |
| жовтий | `color("yellow")` |
| синій | `color("blue")` |
| рожевий | `color("pink")` |
| фіолетовий | `color("purple")` |

> ⚠️ Колір **завжди в лапках**.

**Знайди три `# TODO`** і заміни їх на справжні `color(...)`:
- стебло і листочки → зелений
- пелюстки → будь-який яскравий
- серединка → жовта

Збережи (**Ctrl+S**) і запусти ▶️.

## Крок 4. Експерименти

**Прискор малювання** — додай після `from turtle import *`:
```python
speed(0)
```

**Зміни фон**:
```python
bgcolor("lightblue")
```

**Зафарбуй пелюстки повністю**:
```python
for i in range(6):
    begin_fill()
    circle(25)
    end_fill()
    left(60)
```

**Більше пелюсток** — у циклі заміни `range(6)` на `range(12)`, а `left(60)` на `left(30)`.

# УРОК 2 — Сад 🌻

На минулому уроці ти намалював одну квітку. Сьогодні ми намалюємо **15 квіток** у сітці 3×5 — і не доведеться копіювати код 15 разів!

## Крок 1. Перетвори квітку на функцію

Відкрий свій `flower.py` з минулого уроку.

**Замінимо весь код** — тепер квітка буде «упакована» в функцію `draw_flower(x, y)`. Видали все, що було, і встав це:

```python
from turtle import *
from random import random

speed(0)


def draw_flower(x, y):
    """Малює одну квітку у точці (x, y)."""
    penup()
    goto(x, y - 280)
    setheading(90)
    pendown()

    color("green")
    forward(60)
    circle(20)
    forward(60)
    right(180)
    circle(20)
    right(180)
    forward(60)
    circle(20)
    forward(100)

    right(90)
    color("red")
    for i in range(6):
        circle(25)
        left(60)

    penup()
    goto(x, y - 10)
    color("yellow")
    pendown()
    circle(10)


# --- ТУТ ТИ БУДЕШ ПИСАТИ СВІЙ КОД ---


done()
```

> 🔍 **Що тут нового?** Усі команди малювання тепер всередині `def draw_flower(x, y):`. Це означає: «Python, запам'ятай, як малювати квітку. Я скажу тобі коли і де».

Запусти ▶️. **Має бути порожнє вікно** — це нормально, ми ще не сказали програмі малювати.

## Крок 2. Намалюй одну квітку

Знайди рядок `# --- ТУТ ТИ БУДЕШ ПИСАТИ СВІЙ КОД ---` і додай під ним:

```python
draw_flower(0, 0)
```

▶️ Run. З'явилась одна квітка в центрі.

Спробуй інші координати:
```python
draw_flower(-200, 0)
draw_flower(200, 100)
```

> 💡 **Координати:** `x` — зліва-направо, `y` — знизу-вверх.

## Крок 3. Що таке цикл?

Якщо хочемо **5 квіток у ряд** — нерозумно писати `draw_flower(...)` п'ять разів. Кращий спосіб — **цикл**:

```python
for i in range(5):
    print("Привіт!")
```

Це означає: «**5 разів** виконай те, що всередині». `i` — номер повторення (`0, 1, 2, 3, 4`).

Ти вже бачив цикл — у функції `draw_flower` є:
```python
for i in range(6):
    circle(25)
    left(60)
```
Тут `6 разів` малюється коло з поворотом.

## Крок 4. Завдання — ряд з 5 квіток

Видали попередні `draw_flower(...)` і напиши цикл:

```python
for i in range(5):
    x = -200 + i * 100
    draw_flower(x, 0)
```

**Розгадай формулу:** при `i = 0` → `x = -200`, при `i = 1` → `x = -100`... Між квітками 100 пікселів.

▶️ Run. Маєш отримати 5 квіток у ряд.

> ⚠️ Звертай увагу на **відступи** — рядки всередині циклу мають бути зсунуті праворуч.

## Крок 5. Завдання — сад 3×5

Тепер головне: **3 ряди по 5 квіток**. Для цього потрібен **цикл усередині циклу** — вкладений цикл.

Заміни попередній цикл на цей:

```python
for row in range(3):
    for col in range(5):
        x = -200 + col * 100
        y = -150 + row * 150
        draw_flower(x, y)
```

> 💡 `row` — номер ряду, `col` — номер колонки. Зовнішній цикл повторюється 3 рази, внутрішній — 5 разів. **Разом: 3 × 5 = 15 квіток**.

▶️ Run. Має з'явитись сад! 🌻

**Якщо щось не так:**
- `IndentationError` → перевір відступи. Внутрішній цикл і команди в ньому мають бути зсунуті більше за зовнішній.
- Квітки вилазять за екран → зменш числа `col * 100` та `row * 150` (наприклад, на `90` і `130`).

## Крок 6. Завдання — випадкові кольори 🎨

Зараз усі квітки червоні. Зробимо їх **різнокольоровими**!

### 6.1. Як працюють кольори в Python

Будь-який колір складається з трьох чисел: **скільки червоного, зеленого, синього** (RGB):

| Команда | Колір |
|---------|-------|
| `color(1, 0, 0)` | червоний |
| `color(0, 1, 0)` | зелений |
| `color(0, 0, 1)` | синій |
| `color(1, 1, 0)` | жовтий |
| `color(0.5, 0, 0.5)` | фіолетовий |

Числа від `0` (нічого) до `1` (максимум).

### 6.2. Команда `random()`

Команда `random()` дає випадкове число від 0 до 1 — кожного разу інше:
- `0.7234`, `0.0512`, `0.9981`...

Ми вже додали її на початку файлу: `from random import random`.

### 6.3. Зміни функцію `draw_flower`

У функції знайди рядок:
```python
color("red")
```

І заміни його на:
```python
color(random(), random(), random())
```

▶️ Run. Тепер кожна квітка має **унікальний випадковий колір**! Запусти кілька разів — щоразу новий сад.

# 💡 Шпаргалка

### Команди turtle

| Команда | Що робить |
|---------|-----------|
| `forward(100)` | Уперед на 100 |
| `left(90)` / `right(90)` | Поворот ліворуч / праворуч |
| `circle(25)` | Коло радіусом 25 |
| `goto(x, y)` | Перейти у точку |
| `penup()` / `pendown()` | Підняти / опустити перо |
| `color("red")` або `color(1, 0, 0)` | Змінити колір |
| `bgcolor("lightblue")` | Колір фону |
| `speed(0)` | Найшвидше |

### Цикли

```python
# простий цикл — повторити 5 разів
for i in range(5):
    ...

# вкладений цикл — сітка 3×5
for row in range(3):
    for col in range(5):
        ...
```

### Функція

```python
def draw_flower(x, y):    # визначення
    ...

draw_flower(100, 200)     # виклик
```
