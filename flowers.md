# УРОК 1 — Квітка 🌸

## Крок 1. Створи файл

1. Відкрий **Mu Editor**.
2. Натисни **«New»**.
3. **«Save»** → назва `flower.py`.

## Крок 2. Запусти базовий код

Скопіюй цей код у Mu:

```python
from turtle import *

# --- стебло і листочки ---
penup()
goto(0, -150)
left(90)
pendown()

# TODO: обери колір для стебла і листочків
pensize(5)              # товщина стебла

forward(50)
circle(15)              # лівий листочок
forward(50)
right(180)
circle(15)              # правий листочок
right(180)
forward(50)
circle(15)              # верхній лівий листочок
forward(80)

# --- пелюстки ---
pensize(1)              # повертаємо тонке перо
right(90)
# TODO: обери колір пелюсток
for i in range(6):
    circle(20)
    left(60)

# --- серединка ---
penup()
goto(0, 80)
# TODO: обери колір серединки
pendown()
circle(8)

done()
```

Натисни ▶️ **Run**. Має з'явитись чорна квітка.

> 💡 Закривай вікно з малюнком перед кожним новим запуском.

## Крок 3. Додай кольори

Команда зміни кольору:

```python
color("red")
```

| Колір | Команда |
|-------|---------|
| зелений | `color("green")` |
| червоний | `color("red")` |
| жовтий | `color("yellow")` |
| синій | `color("blue")` |
| рожевий | `color("pink")` |
| фіолетовий | `color("purple")` |

> ⚠️ Колір **завжди в лапках**.

**Знайди три `# TODO`** і заміни їх на справжні `color(...)`:
- стебло і листочки → зелений
- пелюстки → будь-який яскравий
- серединка → жовта

Збережи (**Ctrl+S**) і запусти ▶️.

# УРОК 2 — Сад 🌻

На минулому уроці ти намалював одну квітку. Сьогодні ми намалюємо **6 квіток** у сітці 2×3 — і не доведеться копіювати код 6 разів!

## Крок 1. Перетвори квітку на функцію

Відкрий свій `flower.py` з минулого уроку.

**Замінимо весь код** — тепер квітка буде «упакована» в функцію `draw_flower(x, y)`. Видали все, що було, і встав це:

```python
from turtle import *
from random import random

speed(0)


def draw_flower(x, y):
    """Малює одну квітку у точці (x, y)."""
    # стебло і листочки
    penup()
    goto(x, y - 150)
    setheading(90)
    pendown()

    color("green")
    pensize(5)
    forward(50)
    circle(15)
    forward(50)
    right(180)
    circle(15)
    right(180)
    forward(50)
    circle(15)
    forward(80)

    # пелюстки
    pensize(1)
    right(90)
    color("red")
    for i in range(6):
        circle(20)
        left(60)

    # серединка
    penup()
    goto(x, y + 80)
    color("yellow")
    pendown()
    circle(8)


# --- ТУТ ТИ БУДЕШ ПИСАТИ СВІЙ КОД ---


done()
```

> 🔍 **Що тут нового?** Усі команди малювання тепер всередині `def draw_flower(x, y):`. Це означає: «Python, запам'ятай, як малювати квітку. Я скажу тобі коли і де».

Запусти ▶️. **Має бути порожнє вікно** — це нормально, ми ще не сказали програмі малювати.

## Крок 2. Намалюй одну квітку

Знайди рядок `# --- ТУТ ТИ БУДЕШ ПИСАТИ СВІЙ КОД ---` і додай під ним:

```python
draw_flower(0, 0)
```

▶️ Run. З'явилась одна квітка в центрі.

> 💡 **Координати:** `x` — зліва-направо, `y` — знизу-вверх.

## Крок 3. Що таке цикл?

Якщо хочемо **3 квітки в ряд** — нерозумно писати `draw_flower(...)` тричі. Кращий спосіб — **цикл**:

```python
for i in range(3):
    print("Привіт!")
```

Це означає: «**3 рази** виконай те, що всередині». `i` — номер повторення (`0, 1, 2`).

Ти вже бачив цикл — у функції `draw_flower` є:
```python
for i in range(6):
    circle(20)
    left(60)
```
Тут `6 разів` малюється коло з поворотом.

## Крок 4. Завдання — ряд з 3 квіток

Видали попередній `draw_flower(0, 0)` і напиши цикл:

```python
for i in range(3):
    x = -180 + i * 180
    draw_flower(x, 0)
```

**Розгадай формулу:** при `i = 0` → `x = -180`, при `i = 1` → `x = 0`, при `i = 2` → `x = 180`. Між квітками 180 пікселів — щоб не перетинались.

▶️ Run. Маєш отримати 3 квітки в ряд.

> ⚠️ Звертай увагу на **відступи** — рядки всередині циклу мають бути зсунуті праворуч.

## Крок 5. Завдання — сад 2×3

Тепер головне: **2 ряди по 3 квітки**. Для цього потрібен **цикл усередині циклу** — вкладений цикл.

Заміни попередній цикл на цей:

```python
for row in range(2):
    for col in range(3):
        x = -180 + col * 180
        y = -130 + row * 250
        draw_flower(x, y)
```

> 💡 `row` — номер ряду, `col` — номер колонки. Зовнішній цикл повторюється 2 рази, внутрішній — 3 рази. **Разом: 2 × 3 = 6 квіток**.
>
> 🤔 **Чому між рядами 250, а не 180?** Бо квітка висока: стебло + пелюстки. Менша відстань — і квіти верхнього ряду торкатимуться стебел нижнього.

▶️ Run. Має з'явитись сад! 🌻

**Якщо щось не так:**
- `IndentationError` → перевір відступи. Внутрішній цикл і команди в ньому мають бути зсунуті більше за зовнішній.
- Квіти все ще перетинаються → збільш `180` (між колонками) або `250` (між рядами).

## Крок 6. Завдання — випадкові кольори 🎨

Зараз усі квітки червоні. Зробимо їх **різнокольоровими**!

### 6.1. Як працюють кольори в Python

Будь-який колір складається з трьох чисел: **скільки червоного, зеленого, синього** (RGB):

| Команда | Колір |
|---------|-------|
| `color(1, 0, 0)` | червоний |
| `color(0, 1, 0)` | зелений |
| `color(0, 0, 1)` | синій |
| `color(1, 1, 0)` | жовтий |
| `color(0.5, 0, 0.5)` | фіолетовий |

Числа від `0` (нічого) до `1` (максимум).

### 6.2. Команда `random()`

Команда `random()` дає випадкове число від 0 до 1 — кожного разу інше:
- `0.7234`, `0.0512`, `0.9981`...

Ми вже додали її на початку файлу: `from random import random`.

### 6.3. Зміни функцію `draw_flower`

У функції знайди рядок:
```python
color("red")
```

І заміни його на:
```python
color(random(), random(), random())
```

▶️ Run. Тепер кожна квітка має **унікальний випадковий колір**! Запусти кілька разів — щоразу новий сад.

# 💡 Шпаргалка

### Команди turtle

| Команда | Що робить |
|---------|-----------|
| `forward(100)` | Уперед на 100 |
| `left(90)` / `right(90)` | Поворот ліворуч / праворуч |
| `circle(25)` | Коло радіусом 25 |
| `goto(x, y)` | Перейти у точку |
| `penup()` / `pendown()` | Підняти / опустити перо |
| `pensize(5)` | Товщина лінії |
| `color("red")` або `color(1, 0, 0)` | Змінити колір |
| `bgcolor("lightblue")` | Колір фону |
| `speed(0)` | Найшвидше |

### Цикли

```python
# простий цикл — повторити 3 рази
for i in range(3):
    ...

# вкладений цикл — сітка 2×3
for row in range(2):
    for col in range(3):
        ...
```

### Функція

```python
def draw_flower(x, y):    # визначення
    ...

draw_flower(100, 200)     # виклик
```
