# УРОК 1 — Квітка 🌸

## Крок 1. Створи файл

2. Натисни **«New»**.
3. **«Save»** → назва `flower.py`.

## Крок 2. Запусти базовий код

Скопіюй цей код у Mu:

```python
from turtle import *

# --- стебло і листочки ---
penup()
goto(0, -150)
left(90)
pendown()

# TODO: обери колір для стебла і листочків
pensize(5)              # товщина стебла

forward(50)
circle(15)              # лівий листочок
forward(50)
right(180)
circle(15)              # правий листочок
right(180)
forward(50)
circle(15)              # верхній лівий листочок
forward(80)

# --- пелюстки ---
pensize(1)              # повертаємо тонке перо
right(90)
# TODO: обери колір пелюсток
for i in range(6):
    circle(20)
    left(60)

# --- серединка ---
# TODO: обери колір серединки
circle(8)

done()
```

Натисни ▶️ **Run**. Має з'явитись чорна квітка.

> 💡 Закривай вікно з малюнком перед кожним новим запуском.

# УРОК 2 — Зірка з квіток ⭐

На минулому уроці ти намалював одну квітку — стебло її росло вгору. Сьогодні ми намалюємо **зірку** з 6 квіток: усі починаються в одній точці в центрі, але стебла розходяться у різні боки!

```
       🌸
    🌸     🌸
       ⭐
    🌸     🌸
       🌸
```

## Крок 1. Перетвори квітку на функцію

Відкрий свій `flower.py` з минулого уроку.

**Замінимо весь код** — тепер квітка буде «упакована» в функцію `draw_flower(angle)`. Видали все, що було, і встав це:

```python
from turtle import *
from random import random

speed(0)


def draw_flower(angle):
    """Малює квітку, стебло якої починається з (0, 0) під кутом angle."""
    # переходимо в центр і повертаємось у напрямку angle
    penup()
    goto(0, 0)
    setheading(angle)
    pendown()

    # стебло і листочки
    color("green")
    pensize(5)
    forward(50)
    circle(15)
    forward(50)
    right(180)
    circle(15)
    right(180)
    forward(50)
    circle(15)
    forward(80)

    # пелюстки
    pensize(1)
    right(90)
    color("red")
    for i in range(6):
        circle(20)
        left(60)

    # серединка
    color("yellow")
    circle(8)


# --- ТУТ ТИ БУДЕШ ПИСАТИ СВІЙ КОД ---


done()
```

> 🔍 **Що тут нового?** Усі команди малювання тепер всередині `def draw_flower(angle):`. Це означає: «Python, запам'ятай, як малювати квітку. Я скажу тобі **під яким кутом** її малювати».

Запусти ▶️. **Має бути порожнє вікно** — це нормально, ми ще не сказали програмі малювати.

## Крок 2. Намалюй одну квітку вгору

Знайди рядок `# --- ТУТ ТИ БУДЕШ ПИСАТИ СВІЙ КОД ---` і додай під ним:

```python
draw_flower(90)
```

▶️ Run. Квітка росте вгору з центру.

> 💡 **Чому 90?** У черепашки кути такі: `0` — праворуч, `90` — вгору, `180` — ліворуч, `270` — вниз.

## Крок 3. Спробуй інші напрямки

Замість `draw_flower(90)` напиши:

```python
draw_flower(90)
draw_flower(0)
draw_flower(180)
draw_flower(270)
```

▶️ Run. Тепер 4 квітки розходяться у 4 боки!

## Крок 4. Що таке цикл?

А якщо хочемо **6 квіток** під різними кутами — нерозумно писати `draw_flower(...)` шість разів. Кращий спосіб — **цикл**:

```python
for i in range(6):
    print("Привіт!")
```

Це означає: «**6 разів** виконай те, що всередині». `i` — номер повторення (`0, 1, 2, 3, 4, 5`).

Ти вже бачив цикл — у функції `draw_flower` є:
```python
for i in range(6):
    circle(20)
    left(60)
```
Тут `6 разів` малюється коло з поворотом.

## Крок 5. Завдання — зірка з 6 квіток ⭐

Видали попередні `draw_flower(...)` і напиши **цикл**:

```python
for i in range(6):
    angle = 90 + i * 60
    draw_flower(angle)
```

**Розгадай формулу:**
- При `i = 0` → `angle = 90` (квітка вгору)
- При `i = 1` → `angle = 150`
- При `i = 2` → `angle = 210`
- ...
- При `i = 5` → `angle = 390`, що те саме, що `30°`

Між квітками **60°**, а `360 / 6 = 60` — повне коло поділене на 6 рівних частин!

▶️ Run. Має з'явитись зірка з 6 квіток! ⭐

> ⚠️ Звертай увагу на **відступи** — рядки всередині циклу мають бути зсунуті праворуч.

**Якщо щось не так:**
- `IndentationError` → перевір відступи.
- Квітки розходяться нерівномірно → перевір, що в циклі `range(6)` і кут `i * 60` (бо `360 / 6 = 60`).

## Крок 6. Завдання — більше квіток у зірці

Спробуй зробити зірку з **8 квіток**. Що треба змінити?

- `range(6)` → `range(8)`
- `i * 60` → `i * 45` (бо `360 / 8 = 45`)

```python
for i in range(8):
    angle = 90 + i * 45
    draw_flower(angle)
```

А **12 квіток**? Підказка: `360 / 12 = 30`.

## Крок 7. Завдання — випадкові кольори 🎨

Зараз усі квітки червоні. Зробимо їх **різнокольоровими**!

### 7.1. Як працюють кольори в Python

Будь-який колір складається з трьох чисел: **скільки червоного, зеленого, синього** (RGB):

| Команда | Колір |
|---------|-------|
| `color(1, 0, 0)` | червоний |
| `color(0, 1, 0)` | зелений |
| `color(0, 0, 1)` | синій |
| `color(1, 1, 0)` | жовтий |
| `color(0.5, 0, 0.5)` | фіолетовий |

Числа від `0` (нічого) до `1` (максимум).

### 7.2. Команда `random()`

Команда `random()` дає випадкове число від 0 до 1 — кожного разу інше:
- `0.7234`, `0.0512`, `0.9981`...

Ми вже додали її на початку файлу: `from random import random`.

### 7.3. Зміни функцію `draw_flower`

У функції знайди рядок:
```python
color("red")
```

І заміни його на:
```python
color(random(), random(), random())
```

▶️ Run. Тепер кожна пелюстка зірки має **унікальний випадковий колір**! Запусти кілька разів — щоразу нова зірка.

# 💡 Шпаргалка

### Команди turtle

| Команда | Що робить |
|---------|-----------|
| `forward(100)` | Уперед на 100 |
| `left(90)` / `right(90)` | Поворот ліворуч / праворуч |
| `setheading(90)` | Стати у напрямку 90° (вгору) |
| `circle(25)` | Коло радіусом 25 |
| `goto(x, y)` | Перейти у точку |
| `penup()` / `pendown()` | Підняти / опустити перо |
| `pensize(5)` | Товщина лінії |
| `color("red")` або `color(1, 0, 0)` | Змінити колір |
| `bgcolor("lightblue")` | Колір фону |
| `speed(0)` | Найшвидше |

### Кути в turtle

| Кут | Напрямок |
|-----|----------|
| `0` | праворуч → |
| `90` | вгору ↑ |
| `180` | ліворуч ← |
| `270` | вниз ↓ |

### Цикл і функція

```python
for i in range(6):
    angle = 90 + i * 60
    draw_flower(angle)
```

`i * 60` — рівномірно поділяє коло на 6 частин (`360 / 6 = 60`).
